> **A Premium, Midnight Emerald Financial Powerhouse.**

Welcome to **Sameer**, a state-of-the-art Personal Finance Management System designed to empower you with complete control over your financial life. Built with a focus on **visual excellence**, **premium user experience**, and **intelligent tracking**, Sameer transforms the way you manage your money.

## ✨ Key Features

### 📊 Dynamic Dashboard
Get a high-level overview of your financial health with our **Bento-grid inspired dashboard**. Monitory your total balance, monthly spending vs. budget trends, and stay ahead of upcoming subscription renewals at a glance.

### 🎙️ AI-Powered Voice Entry
Stop manually typing! Add expenses naturally by speaking. 
*   *"Spent ₹15 on coffee"*
*   *"Add 500 for groceries"*
Our **Smart Voice Recognition** automatically categorizes your spending using the Web Speech API.

### 🤝 Group Split (Splitwise-style)
Traveling with friends or sharing an apartment? Create groups, add bills, and let our **Settlement Algorithm** calculate exactly who owes whom. Simple, fair, and transparent.

### 📅 Smart Subscription Tracking
Never get caught off-guard by a recurring bill again. Track all your subscriptions and receive **early warnings** (7-day window) for upcoming billing cycles.

### 🎯 Intelligent Budgeting
Set targets for different categories (Food, Travel, Bills, etc.). Our system monitors your pace and provides **visual alerts** as you approach your limits, helping you stay disciplined.

## 🚀 Getting Started

### Prerequisites
- Node.js (v18+)
- npm or yarn

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/your-username/sameer.git
   cd sameer
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start the development server**
   ```bash
   npm run dev
   ```

4. **Build for production**
   ```bash
   npm run build
   ```
